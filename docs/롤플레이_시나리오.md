# AICC 사업팀 롤플레이 시나리오 (영업 단계별)

> **목적**: AI사업2팀 신규전입/경력직 담당자가 실제 고객 미팅 패턴을 반복 연습하여 IPCC/AICC 도메인 지식과 고객 대응력을 확보하도록 돕는 시뮬레이션 시나리오 라이브러리
>
> **원본 데이터**: 170+건의 실제 고객 미팅 회의록 기반 (대신증권, NH투자증권, 에스원, 우리금융캐피탈, 한화손해보험, KB FCC, CJ, 쿠팡, 로젠택배/TCK, 한국투자증권, 한진택배, 신한투자증권, 하나투어, 소노시니어 등)
>
> **구조**: 2축 매트릭스 — 축1: 영업 단계(A~H, 필수) × 축2: 업종(선택 필터)
>
> **작성일**: 2026-03-28

---
