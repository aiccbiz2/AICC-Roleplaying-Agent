3. 매출 분석 이슈
================================================================================

[현재 파악된 매출]
• 고객 청구 매출: 월 약 7,000만원

[추가 확인 필요 매출]
• 대표번호 인바운드 매출: 월 약 1~2억 추정 (40만 콜 기준)
• 접속료: 사업팀 확인 필요
• 5년 기준 대표번호 매출: 약 10~11억

[매출 귀속 이슈]
• IPT: AICC사업팀에서 손익 검토
• 대표번호(080 등): 전화 사업팀에서 손익 검토
• 통합 손익: 사업팀에서 합산 검증 후 제공

[대표번호 직접 투자비 반영 이슈]
• 확인 필요: 대표번호 손익을 직접 투자비로 반영 가능한지
• 로젠택배 사례에서 안 됐다는 의견 있음 → 재확인 필요
• 안 되면 통합 손익 관점에서 타 항목에 반영

4. 기술적 이슈
================================================================================

[SBC 구축]
• 구성: L2 4대 → L3 일원화
• 예상 비용: 약 9,000만원 (공급가 기준, 유지보수 포함 시 약 4.6억)
• 효율화: 공용 SBC 활용 불가 (KB, 농협생명 등 이미 적재)
• 견적 추가 요청: ITS 컨버전스에 저렴한 단가로 재견적 요청

[전용회선]
• 500메가 4개 → 청약 처리로 투자비 제외 가능성 (사업팀 확인)

[번호 자원] ⚠️ 가장 큰 리스크
• 필요 수량: 약 2만 6,600개
• 현황: 교환기 1대당 1만 3,300개 수용 → 부족 예상
• 신영증권 사례: 6,400개도 빠듯
• 확인 필요: 번호 자원 확보 가능 여부 (사업팀)

[아날로그 라인 연동]
• 현재 구성 불명확 (KT 국사 경유인지 자체 구성인지)
• SKB가 어떻게 처리했는지 확인 필요

[2원 PRI 구성]
• 서울-부산 간 센터 연동용 확인 필요
• 2원 게이트웨이 → SBC 연동 구조 예상

5. 경쟁 환경
================================================================================

[경쟁사 현황]
• KT: MS와 협업 관계, 상무급 방문 영업, 매우 적극적
       → 팀장이 직접 "이거 안 들어오면 안 되냐" 요청할 정도
• SKB: 투자비 경쟁 시 약 1.5배 차이 (불리)

[내부 리스크]
• 19번째 연속 수주 → "LG U+ 너무 밀어주는 거 아니냐" 내부 우려
• 투자비만으로는 경쟁 불리 → 차별화 명분 필요

[전략적 접근]
• 1라운드: 투자비로 탈락하지 않는 게 최우선
• 2라운드: 비슷한 상황에서 차별화 명분 제공
           (IDC 연계, 콜 인프라 사업 연계 등)
