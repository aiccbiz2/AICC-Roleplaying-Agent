================================================================================

• 본 공고: 12월 31일 RFP 공개
• 사전 규격: 12월 초 공개 (STT 700석, 11억)
• 다음정보기술에서 먼저 연락이 왔었고, STT 솔루션을 써보고 싶다고 요청
• 사전규격 기술 부분 검토 후 피드백 완료 (CTO, 사업팀)
• 다음 주 다음정보기술과 미팅 예정

※ 주의: 한전이 "말이 많은 사이트"라 보안 유지 요청이 있었음. 
        내부적으로 100% 공유하지 못한 부분 양해 요청.

2. 사업 구조 확인 (김태홍)
================================================================================

• 예산 40억 미만 → 대기업 입찰 참여 제한
• 이미 작년부터 "마더 업체를 앞에 세우고 뒷단에서 솔루션 납품하는 구조"로 
  김홍기 팀장과 논의해왔음
• 현재 다음정보기술과 별도로 "대표번호 콜센터 PRI 회선 사업"도 진행 중이며, 
  1차 미팅 완료 및 협력 중

3. 협력사 결정 (합의)
================================================================================

[김태홍 팀장 결정]
"다음정보기술이 마더 업체로 나서준다면 200% 감사한 상황. 
 다음정보기술로 가는 것으로 결론."

[추가 확인 사항]
• KT 동향: 지난주 KT에서 다음정보기술에 연락 왔으나, 
           다음정보기술은 KT와 협력 의향 없음 → LG U+ 선호
• 보안 요청: 대외적으로 "LG U+ + 다음정보기술" 협력 관계 최대한 비공개 유지

4. MOU/공문 관련
================================================================================

[양준모 설명]
• LG U+ STT 엔진은 자체 개발이나, 딜리버리/설치는 라피치에서 수행
