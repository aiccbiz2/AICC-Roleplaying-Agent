### 오픈소스 핵심 인사이트

- **AI Interviewer** -- LLM+STT+TTS 3단 파이프라인의 교과서적 구현. OpenAI뿐 아니라 HuggingFace/로컬 모델도 지원하여 온프레미스 배포 가능. 우리 프로젝트에서 음성 대화 구조 참고하기 좋음.
- **FoloUp** -- 면접 생성(JD 기반 질문 자동 생성) -> 링크 공유 -> AI 음성 면접 -> 분석 대시보드 흐름이 우리의 "퀴즈+롤플레이+리포트" 흐름과 유사함.
- **Pipecat / LiveKit Agents** -- 면접 앱이 아니라 **음성 AI 에이전트 프레임워크**. 우리가 직접 도메인 롤플레이를 만들 때 이 위에 구축하면 STT/LLM/TTS 파이프라인을 직접 짜지 않아도 됨. 특히 Pipecat은 Python 기반으로 기존 RAG와 통합이 쉬움.
- **voi-server** -- 에이전트 성격(회의적, 급한, 친근한)을 JSON으로 정의하는 구조가 우리의 고객 페르소나 시뮬레이션에 직접 참고 가능.

## 4. 프로젝트 적용을 위한 종합 분석

### 4-A. 아키텍처 패턴 비교

| 패턴 | 설명 | 해당 솔루션 | 우리 프로젝트 적용성 |
|------|------|------------|---------------------|
| **LLM + STT + TTS 파이프라인** | 음성 입력 -> 텍스트 변환 -> LLM 응답 -> 음성 출력 | AI Interviewer, voi-server | 높음. 고객 통화 롤플레이에 필수 |
| **RAG 기반 시나리오 생성** | 도메인 문서를 벡터화하여 맥락 있는 질문/응답 생성 | Second Nature, FoloUp | **매우 높음**. 이미 rag.py 구현됨. 교육자료 RAG로 질문 생성 |
| **AI 에이전트 프레임워크** | 범용 음성 AI 에이전트 위에 도메인 로직 구축 | Pipecat, LiveKit Agents | 높음. 직접 STT/TTS 구현 대신 프레임워크 활용 |
| **캐릭터/페르소나 시스템** | JSON으로 AI의 성격/말투/반응패턴 정의 | voi-server, Convai, Mindtickle | **매우 높음**. 불만고객/VIP/신규가입자 등 페르소나 필요 |
| **스코어카드/분석** | 응답의 정확성, 톤, 키워드 사용 등 자동 채점 | Hyperbound, Zenarate, Yoodli | 높음. 도메인 용어 사용도, 정확성, 응대 태도 평가 |
| **실제 콜 기반 시뮬레이션** | 실제 고객 통화 녹음을 기반으로 시나리오 생성 | SymTrain, Zenarate | 중간. Discord 교육자료 대신 실제 콜 데이터 확보 시 적용 |

### 4-B. 우리 프로젝트에 가장 참고할 솔루션 TOP 5

1. **Zenarate** -- 컨택센터 전용이라 도메인이 가장 유사. 콜 트랜스크립트 -> 시뮬레이션 자동생성 -> 동일 기준 실전 평가 파이프라인이 이상적 모델.

2. **Hyperbound** -- 10분 내 봇 생성, 맞춤 ICP(고객 프로필) 설정, AI 스코어카드가 우리의 "빠른 시나리오 생성 + 평가" 요구사항과 일치.

3. **Pipecat (오픈소스)** -- Python 기반 음성 AI 프레임워크. 기존 RAG 파이프라인(rag.py)과 통합하여 음성 롤플레이를 구현할 때 가장 실용적인 기반 기술.

4. **voi-server (오픈소스)** -- JSON 기반 에이전트 성격 정의가 우리의 고객 페르소나 시스템 구현에 직접 참고 가능. MIT 라이선스로 자유롭게 활용 가능.

5. **AI Interviewer (오픈소스)** -- LLM+STT+TTS 통합의 가장 깔끔한 레퍼런스 구현. Python/Gradio 기반이라 프로토타입에 즉시 활용 가능.

### 4-C. 권장 기술 스택 조합 (프로젝트 참고)

```
[사용자 음성/텍스트 입력]
        |
   [STT: Whisper / Google STT]
        |
   [RAG: 기존 rag.py + 도메인 문서]  <-- 도메인 용어사전, 교육자료, 시나리오
        |
   [LLM: GPT-4o / Claude]  <-- 페르소나 시스템 프롬프트 (voi-server 패턴)
        |
   [평가 엔진]  <-- 스코어카드 (Hyperbound/Zenarate 패턴)
        |
   [TTS: OpenAI TTS / ElevenLabs]
        |
   [음성/텍스트 출력 + 피드백 대시보드]
```

**프레임워크 선택지:**
- **텍스트 전용 MVP**: 현재 구조(Streamlit/Gradio + RAG) 유지, 롤플레이 프롬프트 추가
- **음성 포함 확장**: Pipecat 또는 LiveKit Agents 위에 도메인 RAG 연결
- **풀 플랫폼**: Next.js + Vapi AI (FoloUp/ai_mock_interviews 패턴)

## 5. 시장 트렌드 요약

1. **컨택센터 훈련 AI 시장 급성장** -- 2025-2026년 Zenarate, SymTrain, Mindtickle 등이 대규모 투자 유치. AI 롤플레이가 기존 스크립트 훈련을 대체하는 추세.

2. **실제 콜 데이터 기반 시뮬레이션이 대세** -- 합성 시나리오보다 실제 고객 통화 데이터를 기반으로 한 시뮬레이션이 효과가 높다는 결과가 반복적으로 보고됨 (SymTrain, Zenarate).

3. **램프타임 50% 단축이 공통 KPI** -- Hyperbound, Zenarate, Mindtickle 모두 신입 직원 온보딩 시간 50% 단축을 핵심 지표로 제시.

4. **음성 AI 프레임워크 성숙** -- Pipecat(7k stars), LiveKit Agents(5k stars) 등 오픈소스 음성 AI 프레임워크가 충분히 성숙하여, 직접 STT/TTS를 구현할 필요 없이 프레임워크 위에서 도메인 로직에 집중 가능.

5. **한국 시장은 채용 면접 중심** -- 뷰인터, 사람인 등 한국 서비스는 채용 면접 평가에 집중. AICC/IPCC 도메인 학습용 롤플레이 솔루션은 아직 블루오션.
