# [교육자료] LG EXAONE 솔루션 소개 Presentation

> **작성일:** 2026.02.13 (금)
> **대상:** 한국투자증권 (IT인프라부, 소비자보호부 등)
> **발표자:** 신재민 (영업), 전병진 팀장 (AX 솔루션 개발)

---

## 1. 개요 및 파트너 경영 체계
LG유플러스는 단순히 모델을 활용하는 것을 넘어, **LG AI연구원과의 독점적 협력 체계**를 통해 차별화된 AI 가치를 제공합니다.

*   **사업 권리:** LG AI연구원에서 개발한 독자 파운데이션 모델 **EXAONE**을 공식적으로 B2B 사업화할 수 있는 권리 보유
*   **기술 공유:** 모델 설계 단계부터 기술적 노하우를 공유받는 **유일한 계열사**
*   **커스텀 역량:** 고객사 요건에 맞춰 **사전학습(Pre-training)부터 운영·유지보수까지** 엔드투엔드(End-to-End) 지원

---

## 2. EXAONE 모델 라인업 및 로드맵
지속적인 업데이트를 통해 글로벌 수준의 성능을 유지하며, 인프라 변경 없는 유연한 업그레이드를 지원합니다.

### 2.1 모델 버전 히스토리
| 버전 | 특징 | 비고 |
| :--- | :--- | :--- |
| **EXAONE 3.5** | 7.8B 등 다양한 효율적 라인업 보유 | 요청 시 딜리버리 가능 |
| **EXAONE 4.0** | **현재 주력 모델 (32B)** | 추론 성능 및 어휘 사전 대폭 확장 |
| **K-EXAONE** | 독자 파운데이션 모델 1차 심사 완료 | 향후 협업 및 적용 가능성 검토 중 |
| **EXAONE 4.5** | **2026년 3월 출시 예정** | 성능 고도화 및 최적화 |

> [!TIP]
> **유연한 업그레이드:** 버전이 올라가도 인프라 스펙은 거의 동일하게 유지되므로, 시스템 아키텍처나 인력 교체 없이 모델만 간편하게 업데이트할 수 있습니다.

### 2.2 핵심 기술 지표 (4.0 32B 기준)
*   **Reasoning:** 복잡한 논리 구조 및 추론 기능 강화
*   **Agentic AI:** MCP(Model Context Protocol) 활용 및 **툴 콜링(Tool Calling)** 기능 최적화
*   **Input Window:** 대규모 컨텍스트 처리를 위한 토큰 사이즈 확장

## 3. 주요 강점 및 차별화 포인트

### 3.1 압도적인 도메인 최적화
*   **한국어 특화:** 글로벌 벤치마크뿐만 아니라 실제 한국어 도메인 활용 시 뛰어난 성능 발휘
*   **폐쇄망 환경 지원:** 온프레미스(On-premise) 등 보안이 중요한 폐쇄망 환경에서 LLM을 구동하려는 고객사에게 최적의 선택지
*   **기술 근본주의:** 스크래치부터 파운데이션 모델을 직접 개발하는 역량을 보유하여, 모델 내부의 근본적인 문제 해결 및 전처리에 강점

### 3.3 금융 도메인 대응 전략
> [!IMPORTANT]
> 금융 데이터의 보안성을 고려하여, LG유플러스는 전용 학습 데이터 증강 기술을 활용합니다.

1.  **사전학습 (Pre-training):** 구매 가능 오픈 데이터와 자체 증강 기술을 통해 **금융 도메인 특화 모델** 생성
2.  **파인튜닝 (Fine-tuning):** 고객사의 고유 상품명, 내부 용어 등을 반영한 2차 미세 조정 수행

## 4. 인프라 및 대량 처리 역량

### 4.1 리소스 유연성
*   **고성능 장비:** NVIDIA H100, A100 등 최신 GPU 환경 최적화
*   **경량화 모델:** 인프라가 제한적인 환경을 위해 **양자화(Quantization) 모델** 보유

### 4.2 대규모 트래픽 처리 (Scalability)
*   **통신사 운영 경험:** 하루 10만 건 이상의 데이터를 처리하는 LG U+ 고객센터 운영 노하우 적용
*   **실시간성 보장:** 상담 어드바이저, QA 등 대량 배치 처리 시에도 **레이턴시 최소화** 가이드 제공 (하루 5만 건 샘플 분석 등)

## 5. Q&A 및 고객 요구사항 분석 (한국투자증권)

### 주요 질문 및 대응
*   **금융권 레퍼런스:** 현재 다수의 금융사와 비공개로 협업 중이며, 특정 서비스 도입부터 전용 LLM 구축까지 다양한 모델로 사업 중
*   **분석 업무 범위 (TA):** 불완전판매 요건 검증 등 TA 항목은 LLM을 통해 수행됨
*   **구현 프로세스:** 단순 모델 제공을 넘어, 금융 실무에 맞는 **AI 비즈니스 프로세스 설계**에 집중
