# [교육자료] LG U+ 상담 어드바이저 솔루션 교육

> **작성일:** 2026.02.13 (금)
> **발표자:** 권태구 PO (고객센터 운영조직), **LGU+ CTO**, 영업 담당 등
> **대상:** 고객 (IT인프라부, 소비자보호부 등)

## 1. 개요 및 조직 역량
LG 유플러스는 단순 기술 공급자를 넘어, **4,500명 규모의 대형 고객센터를 20년 이상 직접 운영**해온 현장 전문가 그룹입니다.

*   **실전 중심 기획:** 현장 상담사의 VOE(Voice of Employee)를 기반으로 실제 업무 효율을 높이는 기능 구현
*   **운영 노하우:** 연간 3,300만 콜 처리 과정에서 축척된 빅데이터와 최신 EXAONE AI 기술의 결합
*   **지속적 개선:** "데일리 피드백 루프" 체계를 통한 상시 품질 고도화

---

## 2. AICC 솔루션 라인업
고객의 문의 시작부터 상담 종료 후 후처리까지, 전 과정에 걸쳐 AI가 개입하여 업무 효율을 극대화합니다.

### 2.1 주요 구성 요소
1.  **AI 콜봇/챗봇:** 단순/반복 문의의 자동화된 신속 해결 (연간 800만 건 처리)
2.  **상담 어드바이저:** 상담사의 실전 대화를 실시간으로 어시스트 (상담 시간 약 20% 감소)
3.  **오토 QA:** 상담 품질의 전수 조사를 통한 코칭 자동화

---

## 3. 핵심 기술 체계: 데일리 피드백 루프
AI 성능의 초기 수치보다 중요한 것은 **"어떻게 빠르게 개선하느냐"**입니다.

*   **프로세스:** 전일 발생한 '의도 분류 오류' 등 오답 데이터를 수집 → 당일 재학습 → 익일 성능 측정
*   **현재 달성 수준:**
    *   의도 분류 정확도: **95%**
    *   코드 추천 정확도: **99%**
    *   상담 요약 정확도: **95%**
*   **자동화 로드맵:** 기존 수동 피드백 과정을 AI로 전환 중 (STT 피드백 오토메이션 하반기 완료 예정)

## 4. 상담 어드바이저 주요 기능 상세

### 4.1 실시간 STT & 지식 추천 (RAG 기반)
*   **고점수 인식률:** 현재 CRR 기준 **97%**의 정확도로 실시간 대화 기록
*   **문맥 기반 지식 추천:** 단순 키워드 매칭이 아닌, **대화의 전체 맥락**을 파악하여 관련 매뉴얼(KMS) 요약 제공
*   **레벨별 개인화:** 신입 상담사에게는 상세 가이드를, 숙련 상담사에게는 필요시에만 정보를 제공하도록 개인별 세팅 가능

> [!TIP]
> **비용 최적화 설계:** 모든 지식을 무분별하게 요약하지 않고, 상담사가 필요로 하는 항목을 클릭할 때만 요약 기능을 작동시켜 API 토큰 비용을 효율적으로 관리합니다.

### 4.2 상담 후처리 자동화 (Post-Call Processing)
*   **자동 요약:** [작업자 / 고객 요청 / 처리 내용]으로 구조화된 고품질 요약문 자동 생성
*   **코드 추천:** 대·중·소 분류에 따른 상담 유형 자동 태깅

### 4.3 오토 QA (상담 품질 평가)
*   **전수 조사 실현:** 1인당 20명을 평가하던 한계를 넘어, 모든 상담의 24가지 항목 자동 평가
*   **코칭 리포트:** 상담 종료 1시간 이내에 감점 요소와 개선안이 포함된 리포트 발행

## 5. 주요 기술 Q&A (고객사 협업 방안)

### Q1. 고객사 내부 매뉴얼(KMS) 연동은 어떻게 하나요?
**LGU+ CTO:** 기존 KMS 시트나 PDF를 LLM이 이해할 수 있는 **RAG(Retrieval-Augmented Generation) 시스템**으로 전환하는 과정이 필요합니다. 저희는 강력한 **문서 파서(Parser) 및 임베딩 기술**을 보유하고 있어, 고객사의 인터페이스 제공 시 별도 시스템 구축 없이도 연동이 가능합니다.

### Q2. 금융 도메인에서의 STT 품질 확보 전략은?
**LGU+ CTO:** 초기 진입 시에는 90% 초반대에서 시작할 수 있으나, 저희만의 **독자적인 STT 엔진과 도메인 특화 피드백 루프**를 통해 최단 시간 내에 97% 이상의 임계치까지 성능을 끌어올립니다.

### Q3. AI 자동 학습(Self-revolving)이 가능한가요?
**LG U+ 전문가:** 오분류 데이터를 자동 수집하여 의도를 재파악하고 매핑하는 **'셀프 리볼빙'** 기술이 구현되어 있습니다. 학습 인프라(GPU) 환경이 마련된다면 고객사 내부에서도 지속적인 자동 학습이 가능하도록 체계를 구축해 드립니다.

### Q4. 짧은 긍정 응답(예: "네")도 인식이 되나요?
**LGU+ CTO:** 상담사 97%, 고객 3% 비율의 극단적인 대화 환경에서도 **음향 모델 강도 조정 및 화자 분리 기술**을 통해 누락 없는 인식이 가능합니다. 민원 이슈 방지를 위해 누락 "0"을 목표로 튜닝합니다.
