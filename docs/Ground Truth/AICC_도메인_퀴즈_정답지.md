## 학습 팁

1. **"CTI 없으면 IPCC 아니다"**: 아키텍처 이해의 핵심. CTI는 PBX-IVR-상담AP를 잇는 두뇌.
2. **스테레오 녹취가 AICC의 전제 조건**: 모노 녹취 환경에서는 AI 화자 분석 자체가 불가능.
3. **RAG + 가드레일 = LLM 안전 아키텍처**: 금융권 AI 도입의 표준 조합.
4. **데일리 체크인**: STT 운영 안정화의 성패를 좌우하는 핵심 프로세스.
5. **솔직한 인정이 신뢰 구축**: LLM 사례가 부족할 때 과장보다 단계적 접근 제안이 효과적.
6. **8:2 평가 구조**: 공공 입찰은 기술 점수가 핵심. PoC + 레퍼런스에 집중.
7. **BPO 현장 대리인**: 도급법 위반(불법파견) 방지를 위한 필수 조건.
